OMS Realtime Workflow

OMS connects the waiter, cashier, and kitchen into one smooth ordering process.

1. Waiter Takes Order

The waiter selects menu items and submits the customer order.

2. Cashier Receives Payment

The cashier receives the order, confirms the total, and processes payment.

3. Kitchen Gets Paid Order

Once paid, the order is sent to the kitchen preparation screen.

4. Ready to Serve

Kitchen marks the order as ready, and the waiter sees that it can be served.

Complete Flow: Waiter → Cashier → Kitchen → Ready to Serve → Waiter Delivery. OMS monitors the full process and tracks all sales and important business information in realtime.

How to Activate OMS

OMS uses an activation key to register and validate your device subscription. To get your official activation key, message the SoftDev PH Facebook page first.

Message SoftDev PH

Activation Steps

  • Download and install the official OMS APK.
  • Open the official Facebook page: SoftDev PH.
  • Send a message requesting your OMS activation key.
  • Provide the required business or device details if requested.
  • Enter the activation key inside the OMS app.
  • Once registered, login and start using OMS.

How to Install and Configure OMS

Since OMS is not yet available on the Play Store, installation is done through a trusted APK file provided directly by the developer.

Download the OMS APK

Tap the Download APK button and get the official OMS APK file from the authorized developer or installer only.

Allow Trusted Installation

On your Android device, allow installation from the browser or file manager used to open the APK.

Install the App

Open the APK file, tap Install, and wait until the installation is completed.

Request Activation Key

Message the official SoftDev PH Facebook page to request and receive your activation key.

Open OMS Settings

Launch OMS and go to the settings page to configure the server or admin host connection.

Login and Start Ordering

After successful activation and connection, login using your assigned role: waiter, kitchen, cashier, or admin.

Important: Android may show a security notice because the app is installed outside the Play Store. This is normal for APK files. Only install OMS from the official provider. Do not install APK files from unknown or unverified sources.
